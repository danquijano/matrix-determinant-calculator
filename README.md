# 📘 Matrix Determinant Calculator

## 📌 Description
This interactive application calculates the determinant of a matrix entered by the user and shows the step-by-step calculation process. Additionally, it provides key theoretical information about determinants.

## 🛠️ Technologies Used
- Python 3
- Streamlit
- NumPy

## 📝 Requirements
- Python 3 installed.
- Required libraries installed:
  ```bash
  pip install streamlit numpy
  
## 📦 Production Build

1. **Clone the repository or copy the source code.**
2. **Save the main file as app.py.**
3. **Run the application with the following command:**

```bash
streamlit run app.py
```
